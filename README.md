## Installation
Clone the repository:
```bash
git clone https://github.com/librecap/librecap.git
```

Go into the librecap directory:
```bash
cd librecap
```

Install the dependencies:
```bash
npm install
```

Build the project:
```bash
npm run build
```

Now you can open the demo.html file in your favorite browser.

## Features
- dark, light and automatic theme
- translations for 107 languages, automatic language detection
- customizable with CSS
- works on mobile devices
- no tracking, no cookies, no analytics, no third party services
- PoW and traditional CAPTCHA
